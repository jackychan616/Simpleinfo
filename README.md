# [Simple Info website](https://simpleinfohk.me)

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Writer platform (v2)

- `/writer` 投稿中心
- `/writer/new` 建立投稿（寫入 Supabase `writer_submissions`）
- `/writer/submissions` 審核管理（pending / approved / rejected）

Setup guide:
- `docs/SUPABASE_SETUP.md`
- `.env.example`
